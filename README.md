# CasaBengala
Prueba tecnica

## Tecnologias Utilizadas
- Aplicacion
  - Java JDK 21
  - Spring Boot 4
  - Hibernate
- IDE
  - IntelliJ IDEA/Visual Code
- Herramientas web
  - HTML
  - CSS
  - Angular
-Base de datos
  -MySql 8.0
-Otros
  -Docker
  -Swagger

## Retos
Uno de los principales retos presentados fue la generacion de la plantilla, por la experiencia que cuento en la parte de diseño se hizo de herramientas externas para poder mejorar el diseño empleado y el uso de IA para pdoer generar el logo y el video empleado en la pagina web.

# Paso Previo
Tener instalado docker en el equipo donde se realizaran pruebas

Clonar el proyecto en el equipo a realizar las pruebas. y abrir una consola sobre la direecion de la carpeta donde se clono el proyecto.

# Instalacion
Para realizar la ejecucion de la aplicacion, se ejecutara el siguiente comando
- docker compose up -d
Con esta instruccion vamos a levantar la base de datos, servicio web y aplicacion web
